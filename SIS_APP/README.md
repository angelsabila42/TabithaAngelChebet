# User’s Guide

## Installation

Latest version of Python

### Dependencies

#### Steps 1: Create an environment I Use macOS/Linux

For Windows Check Link: https://flask.palletsprojects.com/en/stable/installation/#install-flask

mkdir SIS_APP
cd SIS_APP
python3 -m venv .venv

#### Step 2: Activate the environment

. .venv/bin/activate
source .venv/bin/activate

#### Step 3: Install Flask

pip install Flask

#### Step 4: Install Database Flask-SQLAlchemy

Link: https://pypi.org/project/Flask-SQLAlchemy/

pip install Flask-SQLAlchemy
